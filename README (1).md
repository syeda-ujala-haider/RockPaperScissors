
# 🎮 Rock-Paper-Scissors Game

A simple **Rock-Paper-Scissors** game built in **Python** where you play against the computer.  
The computer randomly chooses **Rock (r)**, **Paper (p)**, or **Scissors (s)**, and you have to beat it!  

---

## **📌 Features**
✅ Play Rock-Paper-Scissors against the computer  
✅ Randomized computer choices  
✅ Simple and clean interface  
✅ Continuous gameplay until you decide to **quit**  
✅ Beginner-friendly Python project  

---

## **🕹️ How to Play**
- **r** → Rock 🪨
- **p** → Paper 📄
- **s** → Scissors ✂️
- **q** → Quit the game 🚪

The game follows the classic rules:
- Rock 🪨 beats Scissors ✂️
- Scissors ✂️ beat Paper 📄
- Paper 📄 beats Rock 🪨

